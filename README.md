# Тесты производительности

Сравнение проводилось с двусвязым списком на указателях ([репозиторий](https://github.com/umarnurmatov/mipt-ptr-linked-list)).

## Характеристики системы

| OS | Arch Linux x86_64 |
| :---- | :---- |
| Host | 20K5S16A05 (ThinkPad X270 W10DG) |
| Kernel | Linux 6.17.5-arch1-1 |
| Shell | bash 5.3.3 |
| CPU | Intel(R) Core(TM) i5-6300U (4) @ 3.00 GHz |
| GCC | 15.2.1 20250813 |
| libc | 2.42 |

## Методика тестирования

**Флаги компилятора: -std=c++17 -O2 -march=native**

Измерения проводились с помощью утилиты [hyperfine](https://github.com/sharkdp/hyperfine).   
Процесс был фиксирован на конкретном (4) ядре процессора с помощью утилиты [taskset](https://www.man7.org/linux/man-pages/man1/taskset.1.html).

Перед основными измерениями проводились 2 не учитываемых в статистике для прогрева кэшей (**warmup**).
